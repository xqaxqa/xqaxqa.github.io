💡 开篇类比：导数是什么？

🚗 想象你在开车：

• 你的位置 = 函数 f(x)

• 你的速度 = 导数 f'(x)

• 平均速度 = 总路程 ÷ 总时间（粗略）

• 瞬间速度 = 导数（精确！这一秒有多快）

👉 所以：导数就是"变化的快慢"

📐 主题一：导数的概念（彻底搞懂！）
📖 故事：牛顿是怎么想到的？

牛顿研究"物体运动"时发现：

• 我需要知道每一瞬间的速度，而不只是平均速度

• 比如：子弹飞行时，每一刻的速度都不一样

• 怎么求"这一瞬间"的速度？

💡 牛顿的妙招：

1. 先看"很短时间"内的平均速度

2. 让这个时间无限短（趋近于0）

3. 平均速度就变成"瞬间速度"了！

🎯 核心公式（不用背，理解就行）：
f'(x) = lim(Δx → 0) [f(x + Δx) - f(x)] / Δx

人话翻译：
x变化一点点 → 看y变化多少 → 比值 = 导数

图示：割线（蓝）→ 切线（红），当Δx越来越小时

🎯 记住这3点就够了：

1️⃣ 几何意义：导数 = 切线斜率（曲线上一点的坡度）

2️⃣ 物理意义：导数 = 瞬间速度（这一刻变化多快）

3️⃣ 生活意义：导数 = 敏感度（x动一点，y动多少）

📝 超简单例题：

题目：f(x) = x²，求 f'(3)（在x=3处的导数）

解法1（用定义）：

f'(3) = lim(Δx→0) [(3+Δx)² - 3²] / Δx

= lim(Δx→0) [9 + 6Δx + (Δx)² - 9] / Δx

= lim(Δx→0) [6Δx + (Δx)²] / Δx

= lim(Δx→0) (6 + Δx) = 6

解法2（用公式）：

(x²)' = 2x → f'(3) = 2×3 = 6 ✓

意思：在x=3这个点，x增加1单位，y大约增加6单位

📐 主题二：基本求导公式（像背乘法表一样背！）
💡 类比：就像学骑自行车

• 一开始需要想：怎么保持平衡？怎么踩踏板？

• 熟练后：直接骑，不用想！

• 求导公式也是：一开始需要推导，熟练后直接用！

原函数 f(x) 导数 f'(x) 记忆口诀
xⁿ（幂函数） nxⁿ⁻¹ 指数降1，乘以原指数
sinx（正弦） cosx（余弦） 正变余
cosx（余弦） -sinx（负正弦） 余变负正
eˣ（指数） eˣ（不变！） eˣ最特殊，导数等于自己
lnx（对数） 1/x 对数导数等于1/x
aˣ（指数） aˣ · lna 指数导数 = 自己 × ln(底)
⚠️ 小贴士：

✅ 必须熟练！就像背乘法表，不熟练后面会卡死

✅ 多练！找10道题，每题用不同公式，练到不用想

✅ 常见组合：

  • (u + v)' = u' + v'（和的导数 = 导数的和）

  • (u · v)' = u'v + uv'（乘积法则）

  • (u/v)' = (u'v - uv') / v²（商法则）

📝 综合例题：

题目：f(x) = 3x⁴ - 2sinx + 5eˣ + ln(x+1)，求 f'(x)

解：

f'(x) = (3x⁴)' - (2sinx)' + (5eˣ)' + [ln(x+1)]'

= 3·4x³ - 2cosx + 5eˣ + 1/(x+1)

= 12x³ - 2cosx + 5eˣ + 1/(x+1)

📈 主题三：导数与函数的单调性（超直观！）
📖 故事：爬山 vs 下山

想象你在爬山：

• 上山时：越走越高 → 函数递增 → 导数 > 0

• 下山时：越走越低 → 函数递减 → 导数 < 0

• 山顶/山谷：这一刻不升不降 → 导数 = 0

👉 所以：看导数的符号，就知道函数往哪走！

🎯 核心关系（背下来！）：
f'(x) > 0 → f(x) 单调递增（上山）
f'(x) < 0 → f(x) 单调递减（下山）
f'(x) = 0 → 可能极值点（山顶/山谷）
📋 解题步骤（标准流程）：
1 求定义域
比如：分母不能为0、根号内≥0、对数真数>0
2 求导数 f'(x)
用基本公式、链式法则等
3 解 f'(x) > 0 和 f'(x) < 0
找到导数正负的区间
4 写出单调性
递增区间：f'(x) > 0 的解
递减区间：f'(x) < 0 的解

图示：红线 = 原函数，蓝线 = 导数。导数>0时（蓝线在x轴上方），函数递增

📝 经典例题：

题目：求 f(x) = x³ - 3x 的单调区间

解：

步骤1：定义域 = R（全体实数）

步骤2：求导数

f'(x) = 3x² - 3 = 3(x² - 1) = 3(x-1)(x+1)

步骤3：解 f'(x) > 0 和 f'(x) < 0

f'(x) > 0 → 3(x-1)(x+1) > 0 → x < -1 或 x > 1

f'(x) < 0 → 3(x-1)(x+1) < 0 → -1 < x < 1

步骤4：写出单调性

• 递增区间：(-∞, -1) 和 (1, +∞)

• 递减区间：(-1, 1)

✅ 检验：画图或用计算器验证

⚠️ 常见错误：

❌ 忘记求定义域（比如对数、分式）

❌ 解不等式时，符号方向搞反

❌ 区间端点写成闭区间（一般写开区间）

✅ 建议：画数轴，标根，穿针引线法解不等式

🎯 主题四：极值与最值（山顶 vs 全程最高）
💡 类比：爬山时的"局部最高" vs "全程最高"

🏔️ 极值（局部概念）：

• 你在半山腰的一个小山坡顶 → 这是"局部最高"

• 但不一定是全程最高（可能后面有更高的山）

• 极值 = 某个点比周围都高（或都低）

🏆 最值（全局概念）：

• 全程的最高点 → 这是"全局最高"

• 可能是极值点，也可能是端点

• 最值 = 整个区间的最高（或最低）

对比项 极值 最值 范围 局部（某一点附近） 全局（整个区间） 个数 可能有多个 最多各一个 位置 导数=0的点 或 不可导点 极值点 或 区间端点 必要条件 f'(x) = 0 无
🎯 极值的判定（两种方法）：

方法1：第一充分条件（看导数符号变化）
f'(x) 在 x₀ 左侧 > 0，右侧 < 0 → 极大值
f'(x) 在 x₀ 左侧 < 0，右侧 > 0 → 极小值
f'(x) 在 x₀ 左右同号 → 不是极值

方法2：第二充分条件（看二阶导数）
f'(x₀) = 0 且 f''(x₀) > 0 → 极小值
f'(x₀) = 0 且 f''(x₀) < 0 → 极大值
📋 求最值的标准流程：
1 求导数 f'(x)
2 解 f'(x) = 0，得到驻点
3 求所有极值 + 端点值
4 比较大小，最大的就是最大值，最小的就是最小值

图示：红点 = 极大值，蓝点 = 极小值，绿点 = 端点。最值需要比较所有点的函数值

📝 经典例题：

题目：求 f(x) = x³ - 3x 在 [-2, 3] 上的最大值和最小值

解：

步骤1：求导数

f'(x) = 3x² - 3 = 3(x-1)(x+1)

步骤2：解 f'(x) = 0

x = -1 或 x = 1（两个驻点）

步骤3：求所有候选点的函数值

• f(-2) = (-2)³ - 3(-2) = -8 + 6 = -2（左端点）

• f(-1) = (-1)³ - 3(-1) = -1 + 3 = 2（极大值）

• f(1) = 1³ - 3×1 = 1 - 3 = -2（极小值）

• f(3) = 3³ - 3×3 = 27 - 9 = 18（右端点）

步骤4：比较大小

• 最大值 = 18（在 x = 3 处）

• 最小值 = -2（在 x = -2 和 x = 1 处）
