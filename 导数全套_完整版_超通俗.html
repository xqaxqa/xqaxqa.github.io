📐 主题一：导数的概念（彻底搞懂！）
📖 故事：牛顿是怎么想到的？

牛顿研究"物体运动"时发现：

• 我需要知道每一瞬间的速度，而不只是平均速度

• 比如：子弹飞行时，每一刻的速度都不一样

• 怎么求"这一瞬间"的速度？

💡 牛顿的妙招：

1. 先看"很短时间"内的平均速度

2. 让这个时间无限短（趋近于0）

3. 平均速度就变成"瞬间速度"了！

💡 类比：开车的速度表

• 你看速度表：这一刻显示 60 km/h → 这就是"瞬间速度"

• 但如果只有里程表：总路程 120 km，总时间 2 h → 平均速度 60 km/h

• 导数 = 速度表（瞬间速度）vs 平均速度

🎯 核心公式（不用背，理解就行）：
f'(x) = lim(Δx → 0) [f(x + Δx) - f(x)] / Δx

人话翻译：
x变化一点点 → 看y变化多少 → 比值 = 导数
🎯 记住这3点就够了：

1️⃣ 几何意义：导数 = 切线斜率（曲线上一点的坡度）

2️⃣ 物理意义：导数 = 瞬间速度（这一刻变化多快）

3️⃣ 生活意义：导数 = 敏感度（x动一点，y动多少）

📝 超简单例题：

题目：f(x) = x²，求 f'(3)（在x=3处的导数）

解法1（用定义）：

f'(3) = lim(Δx→0) [(3+Δx)² - 3²] / Δx

= lim(Δx→0) [9 + 6Δx + (Δx)² - 9] / Δx

= lim(Δx→0) [6Δx + (Δx)²] / Δx

= lim(Δx→0) (6 + Δx) = 6

解法2（用公式）：

(x²)' = 2x → f'(3) = 2×3 = 6 ✓

意思：在x=3这个点，x增加1单位，y大约增加6单位

📐 主题二：基本求导公式（像背乘法表一样背！）
💡 类比：就像学骑自行车

• 一开始需要想：怎么保持平衡？怎么踩踏板？

• 熟练后：直接骑，不用想！

• 求导公式也是：一开始需要推导，熟练后直接用！

原函数 f(x) 导数 f'(x) 记忆口诀
xⁿ（幂函数） nxⁿ⁻¹ 指数降1，乘以原指数
sinx（正弦） cosx（余弦） 正变余
cosx（余弦） -sinx（负正弦） 余变负正
eˣ（指数） eˣ（不变！） eˣ最特殊，导数等于自己
lnx（对数） 1/x 对数导数等于1/x
aˣ（指数） aˣ · lna 指数导数 = 自己 × ln(底)
🎯 运算法则（必须熟练！）：

✅ 和差法则：(u ± v)' = u' ± v'

✅ 乘积法则：(u · v)' = u'v + uv'

✅ 商法则：(u/v)' = (u'v - uv') / v²

✅ 链式法则：f(g(x))' = f'(g(x)) · g'(x)

📝 综合例题：

题目：f(x) = 3x⁴ - 2sinx + 5eˣ + ln(x+1)，求 f'(x)

解：

f'(x) = (3x⁴)' - (2sinx)' + (5eˣ)' + [ln(x+1)]'

= 3·4x³ - 2cosx + 5eˣ + 1/(x+1)

= 12x³ - 2cosx + 5eˣ + 1/(x+1)

📈 主题三：导数与函数的单调性（超直观！）
📖 故事：爬山 vs 下山

想象你在爬山：

• 上山时：越走越高 → 函数递增 → 导数 > 0

• 下山时：越走越低 → 函数递减 → 导数 < 0

• 山顶/山谷：这一刻不升不降 → 导数 = 0

👉 所以：看导数的符号，就知道函数往哪走！

🎯 核心关系（背下来！）：
f'(x) > 0 → f(x) 单调递增（上山）
f'(x) < 0 → f(x) 单调递减（下山）
f'(x) = 0 → 可能极值点（山顶/山谷）
📋 解题步骤（标准流程）：
1 求定义域
比如：分母不能为0、根号内≥0、对数真数>0
2 求导数 f'(x)
用基本公式、链式法则等
3 解 f'(x) > 0 和 f'(x) < 0
找到导数正负的区间
4 写出单调性
递增区间：f'(x) > 0 的解
递减区间：f'(x) < 0 的解
📝 经典例题：

题目：求 f(x) = x³ - 3x 的单调区间

解：

步骤1：定义域 = R（全体实数）

步骤2：求导数

f'(x) = 3x² - 3 = 3(x² - 1) = 3(x-1)(x+1)

步骤3：解 f'(x) > 0 和 f'(x) < 0

f'(x) > 0 → 3(x-1)(x+1) > 0 → x < -1 或 x > 1

f'(x) < 0 → 3(x-1)(x+1) < 0 → -1 < x < 1

步骤4：写出单调性

• 递增区间：(-∞, -1) 和 (1, +∞)

• 递减区间：(-1, 1)

🎯 主题四：极值与最值（山顶 vs 全程最高）
💡 类比：爬山时的"局部最高" vs "全程最高"

🏔️ 极值（局部概念）：

• 你在半山腰的一个小山坡顶 → 这是"局部最高"

• 但不一定是全程最高（可能后面有更高的山）

• 极值 = 某个点比周围都高（或都低）

🏆 最值（全局概念）：

• 全程的最高点 → 这是"全局最高"

• 可能是极值点，也可能是端点

• 最值 = 整个区间的最高（或最低）

对比项 极值 最值
范围 局部（某一点附近） 全局（整个区间）
个数 可能有多个 最多各一个
位置 导数=0的点 或 不可导点 极值点 或 区间端点
🎯 极值的判定（两种方法）：

方法1：第一充分条件（看导数符号变化）
f'(x) 在 x₀ 左侧 > 0，右侧 < 0 → 极大值
f'(x) 在 x₀ 左侧 < 0，右侧 > 0 → 极小值

方法2：第二充分条件（看二阶导数）
f'(x₀) = 0 且 f''(x₀) > 0 → 极小值
f'(x₀) = 0 且 f''(x₀) < 0 → 极大值
📋 求最值的标准流程：
1 求导数 f'(x)
2 解 f'(x) = 0，得到驻点
3 求所有极值 + 端点值
4 比较大小，最大的就是最大值，最小的就是最小值
📝 经典例题：

题目：求 f(x) = x³ - 3x 在 [-2, 3] 上的最大值和最小值

解：

步骤1：求导数 f'(x) = 3x² - 3

步骤2：解 f'(x) = 0 → x = -1 或 x = 1

步骤3：求所有候选点的函数值

• f(-2) = -2（左端点）

• f(-1) = 2（极大值）

• f(1) = -2（极小值）

• f(3) = 18（右端点）

步骤4：比较大小

• 最大值 = 18（在 x = 3 处）

• 最小值 = -2（在 x = -2 和 x = 1 处）

🔧 主题五：函数中的构造问题（像拼积木一样！）
📖 故事：为什么需要构造函数？

有时候题目给的条件，不是直接能用的。

比如：已知 f'(x) > f(x)，要证明某个结论

• 直接看：f'(x) 和 f(x) 混在一起，不好处理

• 💡 妙招：构造一个新函数 g(x)，让问题变简单！

💡 类比：做菜时的"调味"

• 原材料（f(x)）→ 不好吃

• 加点调料（构造 g(x) = f(x) · eˣ 或其他）→ 好吃！

• 构造函数 = 找对"调料"，让问题变简单

🎯 常见构造套路（记下来！）：

f'(x) + f(x) > 0 → 构造 g(x) = eˣ · f(x)
f'(x) - f(x) > 0 → 构造 g(x) = f(x) / eˣ
xf'(x) + f(x) > 0 → 构造 g(x) = x · f(x)
要证 f(x) > g(x) → 构造 h(x) = f(x) - g(x)
📋 构造函数的思路（3步）：
1 观察题目条件
看看有什么特征？f'(x) 和 f(x) 的关系？
2 选择合适的构造
对照上面的表格，选一个
3 验证构造是否有效
求 g'(x)，看符号是否确定
📝 经典例题：

题目：已知 f'(x) > f(x) 对任意 x 成立，且 f(0) = 1，证明：f(x) > eˣ

解：

构造：g(x) = f(x) / eˣ

求导：g'(x) = [f'(x) - f(x)] / eˣ

• ∵ f'(x) - f(x) > 0，eˣ > 0

• ∴ g'(x) > 0 → g(x) 单调递增

证明：

要证：f(x) > eˣ

⇔ g(x) = f(x)/eˣ > 1

∵ g(x) 递增，且 g(0) = f(0)/e⁰ = 1

∴ 当 x > 0 时，g(x) > g(0) = 1 ✓

♾️ 主题六：恒成立与存在性问题（搞清楚∀和∃！）
📖 故事：全班考试 vs 至少一个及格

♾️ 恒成立（∀，forall）：

• 全班所有人考试都 > 60 分 → 恒成立

• 翻译：最小值 > 60

∃ 存在性（∃，exists）：

• 班里至少有一个人考试 > 60 分 → 存在性

• 翻译：最大值 > 60

👉 所以：恒成立看最小值，存在性看最大值

问题类型 数学语言 转化方法
恒成立 ∀x ∈ D, f(x) > a f(x)的最小值 > a
恒成立 ∀x ∈ D, f(x) < a f(x)的最大值 < a
存在性 ∃x ∈ D, f(x) > a f(x)的最大值 > a
存在性 ∃x ∈ D, f(x) < a f(x)的最小值 < a
🎯 含参数的恒成立问题（重点！）：

解法1：分离参数（推荐！）
• 把 a 单独放一边：a > g(x) 或 a < g(x)
• 求 g(x) 的最值 → 得到 a 的范围

解法2：分类讨论
• 把 a 当作常数，求 f(x) 的最值
• 让最值满足题目条件 → 解 a
📋 分离参数的标准流程：
1 移项
把所有含 a 的项放一边，含 x 的放另一边
2 分离
得到 a > g(x) 或 a < g(x)（注意除负数要变号！）
3 求最值
用导数求 g(x) 的最大值或最小值
4 写答案
a > max(g(x)) 或 a < min(g(x))
📝 经典例题：

题目：∀x > 0, ax + 1/x ≥ 2 恒成立，求 a 的取值范围

解：

分离参数：

ax + 1/x ≥ 2

→ a ≥ (2 - 1/x) / x = 2/x - 1/x²（注意x > 0，不变号）

令 t = 1/x（t > 0），则：

a ≥ 2t - t² = -(t-1)² + 1

求最大值：

• -(t-1)² + 1 是开口向下的抛物线

• 当 t = 1 时，取最大值 1

结论：a ≥ 1

✅ 主题七：利用导数证明不等式（比大小！）
📖 故事：比身高

要证明：张三比李四高

• 方法1：直接量身高，比较数字

• 方法2：让张三和李四站一起，看谁高

• 方法3：证明张三的身高 - 李四的身高 > 0

👉 证明不等式 = 证明"左边 - 右边 > 0"

🎯 核心思路（3种方法）：

方法1：构造函数法（最常用！）
• 构造 g(x) = f(x) - h(x)（左 - 右）
• 证明 g(x) 的最小值 ≥ 0（或 > 0）

方法2：值域法
• 求左边函数的最大值 M
• 求右边函数的最小值 m
• 证明 M ≤ m

方法3：放缩法
• 用已知不等式（如 eˣ ≥ x+1）放缩
• 证明放缩后的不等式
📋 构造函数法的标准流程：
1 构造 g(x) = 左 - 右
把不等式右边移项到左边
2 求定义域
确定 x 的范围
3 求 g'(x)，分析单调性
找到 g(x) 的最小值点
4 证明 g(x)的最小值 ≥ 0
如果最小值 ≥ 0，则 g(x) ≥ 0 恒成立
📝 经典例题1：证明 eˣ > x + 1 (x ≠ 0)

解：

步骤1：构造

g(x) = eˣ - x - 1

步骤2：求导

g'(x) = eˣ - 1

• 当 x > 0 时，eˣ > 1 → g'(x) > 0 → g(x) 递增

• 当 x < 0 时，eˣ < 1 → g'(x) < 0 → g(x) 递减

• ∴ x = 0 是极小值点（也是最小值点）

步骤3：证明

g(x)的最小值 = g(0) = e⁰ - 0 - 1 = 0

∵ x ≠ 0，∴ g(x) > g(0) = 0

→ eˣ - x - 1 > 0

→ eˣ > x + 1 ✓

💡 记住这2个经典不等式（常用！）：

✅ eˣ ≥ x + 1（当且仅当 x = 0 时取等）

✅ lnx ≤ x - 1（当且仅当 x = 1 时取等）

这两个可以用来放缩，证明更复杂的不等式！

🔴 主题八：利用导数研究函数的零点（穿过x轴！）
📖 故事：找路口

函数的零点 = 函数图像与 x 轴的交点

• 就像：你从A点走到B点，一定会经过某些路口

• 如果 f(A) > 0，f(B) < 0，那么中间一定有零点！

👉 零点定理：f(a)·f(b) < 0 → ∃c∈(a,b), f(c)=0

🎯 零点问题的3种类型：

类型1：证明零点存在（零点定理）
• 找到 a, b 使得 f(a)·f(b) < 0
• 由零点定理，存在 c∈(a,b) 使得 f(c)=0

类型2：求零点个数（分析单调性）
• 求导数，分析单调性
• 画草图，看穿过x轴几次

类型3：已知零点个数，求参数范围
• 分析函数性质（单调性、极值）
• 让极值的符号满足条件
📋 求零点个数的标准流程：
1 求定义域
确定 x 的范围
2 求导数 f'(x)，分析单调性
找到递增、递减区间
3 求极值
找出所有极值点，计算极值
4 画草图，判断零点个数
• 极值的符号？
• 当 x→±∞ 时，f(x)→？
• 结合单调性，画出草图
📝 经典例题：求 f(x) = x³ - 3x + 1 的零点个数

解：

步骤1：定义域 = R

步骤2：求导，分析单调性

f'(x) = 3x² - 3 = 3(x-1)(x+1)

• f'(x) > 0 → x < -1 或 x > 1 → 递增

• f'(x) < 0 → -1 < x < 1 → 递减

步骤3：求极值

• 极大值：f(-1) = 3 > 0

• 极小值：f(1) = -1 < 0

步骤4：画草图，判断零点个数

• 当 x→-∞ 时，f(x)→-∞

• 在 (-∞, -1) 递增，f(-1) = 3 > 0 → 1个零点

• 在 (-1, 1) 递减，f(-1)=3>0，f(1)=-1<0 → 1个零点

• 在 (1, +∞) 递增，f(1)=-1<0，f(x)→+∞ → 1个零点

✅ 结论：f(x) 有 3个零点

🎭 主题九：隐零点与极值点偏移（高难度！）
📖 故事：猜数字游戏

🎭 隐零点：

• 你知道有个人，但不知道他叫什么

• 你知道他的特征（比如：f'(x₀) = 0）

• 虽然解不出 x₀，但可以用他的特征来推导！

🎯 极值点偏移：

• 正常情况下，极值点应该在正中间

• 但有时候会"偏移" → 证明它不在中间

🎯 隐零点的处理（3步）：

步骤1：设 f'(x₀) = 0（不解出 x₀）
步骤2：由 f'(x₀) = 0，得到 x₀ 满足的方程
步骤3：用这个方程化简，证明结论

💡 关键：虽然不知道 x₀ 的具体值，但知道它满足什么关系！
📝 隐零点经典例题：

题目：f(x) = eˣ - ax，证明：当 a > e 时，f(x) 有两个零点

解：

步骤1：求导

f'(x) = eˣ - a

步骤2：设隐零点

令 f'(x₀) = 0 → eˣ₀ = a → x₀ = lna

步骤3：分析极值

• 当 x < x₀ 时，eˣ < a → f'(x) < 0 → 递减

• 当 x > x₀ 时，eˣ > a → f'(x) > 0 → 递增

• ∴ x₀ 是极小值点

步骤4：证明有两个零点

• 极小值 f(x₀) = eˣ₀ - ax₀ = a - a·lna = a(1 - lna)

• ∵ a > e → lna > 1 → 1 - lna < 0 → f(x₀) < 0

• 又 ∵ 当 x→-∞ 时，f(x)→-∞；当 x→+∞ 时，f(x)→+∞

• ∴ f(x) 在 (-∞, x₀) 和 (x₀, +∞) 各有 1个零点

✅ 结论：当 a > e 时，f(x) 有两个零点

💡 最后的话：

✅ 隐零点 = 知道它存在，但解不出来 → 用方程关系

✅ 极值点偏移 = 证明极值点不在正中间 → 构造对称函数

✅ 难度较高 = 需要多练，熟悉套路

✅ 高考/考研重点 = 经常考，要掌握！
