📐 主题一：导数的概念与运算
👤 账号：导数（Derivative）
📡 信息源
牛顿、莱布尼茨、物理学中的瞬时速度问题
👥 关注列表
极限、函数、切线、变化率
👍 点赞记录
瞬间、斜率、敏感度、Δx→0
📑 收藏公式
f'(x) = lim(Δx→0) [f(x+Δx)-f(x)]/Δx
基本求导公式：
(xⁿ)' = nxⁿ⁻¹    (sinx)' = cosx    (eˣ)' = eˣ
(lnx)' = 1/x    (cosx)' = -sinx    (aˣ)' = aˣlna
📝 示例：
求 f(x) = x³ + 2x² - 5x + 3 的导数
解：f'(x) = 3x² + 4x - 5
📈 主题二：导数与函数的单调性
🔗 共同好友：函数单调性
核心关系
f'(x) > 0 → 函数单调递增
f'(x) < 0 → 函数单调递减
🎯 调查方法
1. 求定义域
2. 求导数 f'(x)
3. 解 f'(x) > 0 和 f'(x) < 0

红线：原函数 | 蓝线：导数 | 导数>0时函数递增

📝 示例：
f(x) = x³ - 3x 的单调性？
f'(x) = 3x² - 3 = 3(x²-1)
f'(x) > 0 → x < -1 或 x > 1 (递增)
f'(x) < 0 → -1 < x < 1 (递减)
🎯 主题三：导数与函数的极值、最值
🏔️ 账号：极值与最值
👤 极值
局部最高点/最低点
必要条件：f'(x) = 0
充分条件：f''(x) 的符号
👤 最值
全局最高点/最低点
比较：极值 + 端点值
极值的判定：
第一充分条件：f'(x) 在 x₀ 左右变号 → 极值
第二充分条件：f''(x₀) > 0 → 极小；f''(x₀) < 0 → 极大

红点：极值点 | 导数为0的点不一定是极值！

🔧 主题四：函数中的构造问题
🏗️ 账号：构造函数
🎯 常见构造
1. 构造 g(x) = f(x) - kx
2. 构造 g(x) = f(x) / x
3. 构造 g(x) = eˣf(x)
4. 构造 g(x) = f(x) - f'(x)
💡 构造思路
观察目标式
利用导数关系
转化为单调性/极值问题
📝 经典题：
已知 f'(x) > f(x)，证明 f(x)/eˣ 递增
构造 g(x) = f(x)/eˣ
g'(x) = [f'(x) - f(x)]/eˣ > 0 ✓
♾️ 主题五：恒成立、存在性问题
🔍 账号：恒成立 vs 存在性
♾️ 恒成立
∀x ∈ D, f(x) > 0
→ 求 f(x) 的最小值 > 0
∃ 存在性
∃x ∈ D, f(x) > 0
→ 求 f(x) 的最大值 > 0
解题步骤：
1. 分离参数（如可能）
2. 构造函数值域问题
3. 求最值 → 得到参数范围
📝 示例：
∀x > 0, ax + 1/x ≥ 2 恒成立，求a的范围
解：a ≥ (2 - 1/x)/x = 2/x - 1/x²
令 t = 1/x, 则 a ≥ 2t - t² = -(t-1)² + 1
max = 1 → a ≥ 1
✅ 主题六：利用导数证明不等式
📝 账号：不等式证明
🎯 方法一：构造函数
构造 g(x) = f(x) - h(x)
证明 g(x) 的极值 ≥ 0 或 ≤ 0
🎯 方法二：值域法
求左式的最大值
求右式的最小值
证明 max(LHS) ≤ min(RHS)
📝 经典题：
证明：x > 0 时，x > ln(x+1)
构造 g(x) = x - ln(x+1)
g'(x) = 1 - 1/(x+1) = x/(x+1) > 0
g(x) 递增 → g(x) > g(0) = 0 ✓
🔴 主题七：利用导数研究函数的零点
🎯 账号：零点问题
🔴 零点定理
f(a)·f(b) < 0
→ ∃c ∈ (a,b), f(c) = 0
🎯 解题步骤
1. 确定定义域
2. 分析单调性（求导）
3. 找零点存在区间
4. 确定零点个数

函数穿过x轴的点 = 零点

📝 示例：
f(x) = x³ - 3x + 1 的零点个数？
f'(x) = 3x² - 3 = 3(x²-1)
单调性：(-∞,-1)↑, (-1,1)↓, (1,+∞)↑
f(-1) = 3 > 0, f(1) = -1 < 0
→ 3个零点
🎭 主题八：隐零点与极值点偏移问题
🎭 账号：隐零点与偏移
🎭 隐零点
导数 f'(x) = 0 的根无法显式解出
但知道它存在！
用 f'(x₀) = 0 建立关系
🎯 极值点偏移
对称性问题：f(x) = f(x₀ + t) 与 f(x₀ - t)
证明极值点不在正中央
隐零点的处理：
1. 设 f'(x₀) = 0（不解出 x₀）
2. 用 x₀ 满足的方程化简
3. 构造单变量函数证明
📝 经典题：
f(x) = eˣ - ax，证明有两个零点时极值点在中间
设 f'(x₀) = eˣ₀ - a = 0 → x₀ = ln(a)
分析 f(ln a) 的符号 → 完成证明
